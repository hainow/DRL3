## Setup

To install the packages using pip and a virtualenv run the following
commands:

```
cd deeprl_hw3_src
virtualenv hw3-env
source hw3-env/bin/activate
pip install -U -r requirements.txt
```

Make sure you are on Python 3.5.2.
